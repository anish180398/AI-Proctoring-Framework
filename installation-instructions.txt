1. Download & Install [Python 3.9.6], on your PC.
**Don't forget to add the interpreter to your system's Path. - [select the check box on the installation Window]

2. Download Visual Studio 2019 [Community Edition], and in the installation window, select "Desktop tools for C++" & "Python development & debugging" options, [Required size would be around 8.2 GB]
**Important for the project to run on PC

3. Check if python is installed successfully using [on cmd]:
	
	python --version

4. Install virtual envoirnment package.
	
	pip install virtualenv

**To read more on this : https://www.geeksforgeeks.org/python-virtual-environment/

5. Clone the git repository for the project using:

	git clone https://github.com/deepanshu-Raj/AI-Proctoring-Framework.git
	
   Or, download the project in the .zip format

6. Go inside this project directory using cd command in "cmd". Once inside the folder, create a virtual envoirnment for your pc using :
	
	virtualenv <name_of_virtual_env>

7. Now activate the virtual envoirnment, using command :
	
	source <name_of_virtual_env>/Scripts/activate

**once done, you'll see the name of your virtual envoirnment in parantheses in your cmd.

8. For installing the dependencies for the project, use [on cmd] :

	pip install -r requirements.txt
